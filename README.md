# CompAppsProject
Fitness Data and Health Conditions - Computer Applications for BME

Team Members: Shruti Likhite, Ana Rincon, Kelly Zhao, Evangeline Sullivan

Our Project utilizes fitness tracker data to create a supervised machine learning model that is able to predict the presence of health conditions based on the presented health metrics. Our model first identified the most important features from the data. Utilizing this data, we created a decision tree model that utilizes these features to predict the presence of asthma, diabetes, or hypertension
